# Документация по гиту
git status - список файлов и их статус


# Оформление коммитов
feat: новая функция или фича
Пример: feat: добавлен ввод имени игрока

fix: исправление бага
Пример: fix: исправлена утечка памяти

docs: изменения в доках
Пример: добавлен новый раздел в документацию

chore: рутинные задачи
Например: создание проекта cargo или что-то такое
